> 在此记录一下我觉得比较好用的个人项目技术栈。

## 推荐技术栈

{{< tabs "center" >}}
<!-- tab 前端开发 -->

> 这里的前端指的是：Web端、桌面端、移动端

1. React + Typescript + Tailwind CSS + Shadcn UI + Vite + Tauri

这是一套「前端极速开发 + 桌面原生体验」的黄金技术栈，专门用来做轻量、美观、跨平台（Windows/macOS/Linux/Android/iOS）、启动超快、体积极小的桌面软件。

**不使用Tauri，则是开发Web端应用。**

- [Tauri](https://tauri.app/): 把网页打包成跨平台软件

- [Vite](https://vitejs.cn/): 前端构建工具

- [React](https://zh-hans.react.dev/learn): 前端框架

- [Typescript](https://www.typescriptlang.org/docs/): 前端类型检查

- [Tailwind CSS](https://www.tailwindcss.cn/): 前端原子化CSS框架

- [Shadcn UI](https://www.shadcn.com.cn/docs): 可直接复制源码的前端组件库

<!-- tab 小程序开发 -->

1. React + Taro + Tailwind CSS / 小程序原生框架

真的很推荐前端使用react框架，个人感觉比vue更方便！！！

<!-- tab 后端开发 -->

1. Cloudflare Workers + Hono + Drizzle ORM + D1 + Supabase

Cloudflare 赛博活佛！只需要买个域名，就能部署免费的后端服务！！！

这套技术栈对于个人开发者，极轻微量使用，完全足够！！！
